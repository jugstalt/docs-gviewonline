gView Commandline Tools
=======================

Here are some command line tools for *gView GIS* to automate recurring operations.
The command line tools are located in the installation path in the directory ``cmd``. This absolute path must be added the
commands shown here or a reference in the ``PATH`` environment variable can be made.

.. note::
   all *gView Kommandozeilen Tools* starts with the ``gView.Cmd.*`` prefix.
