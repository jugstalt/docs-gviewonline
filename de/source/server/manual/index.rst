Benutzerhandbuch
================

Der *gView MapServer* ist eine Webanwendung und bietet eine einfache Web-Oberfläche zum Verwalten 
und Durchsuchen der Dienste. Hier wird vorausgesetzt, dass der *gView MapServer* erfolgreich installiert 
wurde (inkl. Postinstallation).

Im Abschnitt :ref:`examples` wird im Kapitel :ref:`publish-map-service-example` gezeigt, wie Dienste über 
die Web-Oberfläche von *gView MapServer* publiziert werden.
Dort werden weitere Konzepte von *gView MapServer* beschrieben, die einen Großteil der Bedienung abdecken.
