Benutzerhandbuch
================

Der *gView MapServer* ist eine WebAnwendung und bietet eine einfache WebOberfläche zum Verwalten und Durchsuchen der Dienste.
Hier wird vorausgesetzt, das *gView MapServer* erfolgreich installiert wurde (inkl. Post-Inatallation).

Im Abschnitt :ref:`examples` wird in Kapitel :ref:`publish-map-service-example` wie Dienste über die WebOberfläche von *gView MapServer* publiziert werden.
Dort werden ich noch weiter Konzepte von *gView MapServer* beschreiben, die einen Großteil der der Bedienung abdecken.
